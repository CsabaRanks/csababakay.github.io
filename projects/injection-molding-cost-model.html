Injection Molding Cost Model

Cost Engineering Parametric Modeling Manufacturing

1. Problem

Fast and reliable cost estimation for injection molded parts is often required early in sourcing and value engineering. In practice, most part information is available, while cycle time is frequently uncertain or missing.

2. Approach

A parametric cost model decomposes total part cost into a small number of interpretable components and uses proxy variables to estimate the missing cycle-time-dependent share.

3. Cost Decomposition

The model uses the following structure:

Cost = Material + Machine + Labor + Overhead

  • Material: mass × material price (+ scrap factor if needed).
  • Machine: estimated cycle time × machine rate (machine class selected from part requirements).
  • Labor: handling and setup proxies (e.g., lot size assumptions).
  • Overhead: defined allocation logic (kept configurable).

4. Feature Vector

Each part is represented as a feature vector enabling consistent benchmarking and model calibration. Example features (non-exhaustive):

  • part mass / material
  • projected area / thickness class
  • machine class proxy (clamp force / shot weight class)
  • cavitation assumption (single / multi-cavity)
  • cycle-time proxy variables (geometry complexity, cooling dominance indicator)

5. Cycle Time Handling (Proxy Concept)

Instead of requiring an exact cycle time, the model estimates a cycle-time proxy derived from part and process indicators. The proxy is designed to be:

  • monotonic in relevant drivers (e.g., thicker sections → longer cooling),
  • stable across similar parts,
  • calibratable using historical quotations / awards.

6. Validation

The model is validated against a held-out set of historical part cost data and evaluated with an accuracy target of >95% (definition depending on the chosen metric, e.g., relative error band).

7. Implementation Notes

  • Designed for fast usage in procurement and value engineering workflows.
  • Configurable parameter tables (rates, overhead, material) to adapt to different plants/regions.
  • Tooling costs excluded in the current scope.
