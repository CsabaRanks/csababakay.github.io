Injection Molding Cost Model

1. Problem Definition

Accurate cost estimation for injection molded parts under incomplete cycle time information.

2. Modeling Approach

Parametric cost model based on material cost, machine rate estimation, geometry-dependent processing time approximation, and overhead allocation.

3. Feature Vector Representation

Each part is represented as a feature vector including: mass, material type, machine class, cavity count, estimated cycle proxy variables.

4. Mathematical Structure

Cost = Material + Machine + Labor + Overhead

5. Validation

Validated against historical procurement data. Achieved >95% accuracy.

6. Implementation

Implemented in Excel-based parametric engine with modular structure.
